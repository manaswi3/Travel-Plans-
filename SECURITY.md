# Security Policy

## Supported Versions

Currently, the `main` branch of this repository is supported with security updates.
If you fork the project or use an older clone, please ensure you sync with `main` frequently to receive security patches.

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability, we appreciate your help in disclosing it to us in a responsible manner.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email or directly message the project maintainer through private channels.

### Process

1. Report the vulnerability privately. Include steps to reproduce the issue, the potential impact, and any mitigation ideas you might have.
2. We will acknowledge receipt of your vulnerability report within 48 hours.
3. We will work with you to understand the problem, develop a fix, and coordinate the public disclosure.
4. We will notify you when a fix is released.

Thank you for helping keep this project safe!
